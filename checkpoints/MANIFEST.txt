# ADMET ArcMol checkpoints — where to get weights

Trained weights are **not** committed to Git. Download the bundle from **Zenodo** and extract into this repository’s **`checkpoints/`** directory (repo root), so each task is `checkpoints/<task_name>/` with a matching pair of `*.bundle.pt` and same-stem `*.pth`.

- DOI: 10.5281/zenodo.18972759
- URL: https://doi.org/10.5281/zenodo.18972759

Unpack archives **into** `checkpoints/` (preserve task folder names). Then run the ADMET notebook or `src/predict_arcmol_from_fp_pkls.py`.
