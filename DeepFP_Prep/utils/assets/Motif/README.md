# Motif (GROVER) — checkpoint for DeepFP_Prep

## Expected file

```text
Motif/model.pth
```

On the GraphMVP shared Drive, the file is named **`Motif.pth`**. **Rename** it to **`model.pth`** and place it here.

## Download

1. Open the [GraphMVP Google Drive folder](https://drive.google.com/drive/folders/1uPsBiQF3bfeCAXSDd4JfyXiTh-qxYfu6?usp=sharing).
2. Download **`Motif.pth`** (Grover / Motif pretraining).
3. Save as **`Motif/model.pth`** under `utils/assets`.
